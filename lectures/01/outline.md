## Overview

A crash course in Python tailored for students coming from a C programming background. We start from the basics — syntax differences, control flow, and data structures — then build up to NumPy vectors, matrices, and Matplotlib visualizations, laying the groundwork for every ML lecture that follows.

## Learning Objectives

By the end of this lecture you will be able to:

1. **Write** Python code in Google Colab and understand the interpreter workflow.
2. **Translate** familiar C constructs (types, loops, arrays) into idiomatic Python.
3. **Use** lists, dictionaries, and list comprehensions effectively.
4. **Manipulate** vectors and matrices with NumPy — including shapes, reshaping, transpose, and multiplication.
5. **Visualize** data using Matplotlib plots, histograms, and interactive widgets.

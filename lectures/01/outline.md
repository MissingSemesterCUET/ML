## Overview

An exciting introduction to the world of Machine Learning and Deep Learning, followed by a fast-paced Python crash course tailored specifically for students coming from a C programming background. 

## Learning Objectives

By the end of this lecture you will be able to:

1. **Describe** what Machine Learning and Deep Learning are and identify their exciting real-world applications.
2. **Write** Python code in Google Colab and understand the interpreter workflow.
3. **Translate** familiar C constructs (types, loops, arrays) into idiomatic Python.
4. **Use** lists, dictionaries, and list comprehensions effectively.
5. **Define** functions in Python and understand scope and return values.
